# Week 5 &mdash; Manifolds and Riemannian Geometry

**Focus:** Tangent spaces, the Laplace&ndash;Beltrami operator, and gradient flow.

## Contents

- [`theory.ipynb`](theory.ipynb) &mdash; mathematical foundations and derivations.
- [`implementation.ipynb`](implementation.ipynb) &mdash; practical Python code, empirical verification, and exercises.

## Topics Covered

1. Smooth manifolds, charts, and tangent spaces.
2. The Riemannian metric: length, geodesics, gradient, volume.
3. The Laplace&ndash;Beltrami operator and the manifold Fourier basis.
4. The discrete cotangent Laplacian on meshes.
5. Heat diffusion, gradient flow, and the Heat Kernel Signature.

## Requirements

NumPy, SciPy (cotangent Laplacian and heat diffusion from scratch).

## How to Use

Work through `theory.ipynb` first to build the conceptual and mathematical foundation, then open `implementation.ipynb` to see the ideas realised in code. Each implementation notebook ends with exercises that extend the material.
